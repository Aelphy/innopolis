	\centerline{\LARGE Theorem 1}

	\bigskip
			
If h(n) is consistent, A* using GRAPH-SEARCH is optimal
\bigskip

1) By definition the cost not decreases during all the path.

2) Let us imagine, that theorem doesn't work. So there is optimal path to target node and nonoptimal through the point A.

\begin{center}
	\begin{tikzpicture}[scale=.8,auto=left,every node/.style={circle,fill=blue!20}]
  \node (n1) at (6,5){start};
  \node (n2) at (6,-3){target};
  \node (n3) at (3,1){A};

  \foreach \from/\to in {n1,n1/n2,n2/n3,n3/n1}
    \draw (\from) -- (\to);

\end{tikzpicture}
\end{center}		

Imagine, that algorithm chooses nonoptimal path.
But it cannot be because the cost is not decreasing and f(n) is consistent.

f(start) = g(start) + h(start) > f(A) = g(A) + h(A)
